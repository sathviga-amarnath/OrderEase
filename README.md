# 🍽️ OrderEase - AI Personal Concierge Agent

> Your intelligent personal concierge for food ordering and shopping in Chennai!

## 🌟 Overview

OrderEase is an AI-powered personal concierge agent built using Google's Gemini AI. It helps users discover restaurants, order food, shop for products, and track orders — all through a simple conversation!

## ✨ Features

- 🔍 **Smart Restaurant Search** — Find restaurants by cuisine, area, and rating
- 🛒 **Product Shopping** — Search and compare products across categories
- 🧠 **Personalized Recommendations** — Remembers your preferences and suggests accordingly
- 📦 **Order Tracking** — Track your orders in real time
- 💬 **Natural Language** — Just talk naturally, no commands needed!
- 📍 **Chennai Focused** — Optimized for Chennai but works globally

## 🛠️ Tech Stack

- **AI Model:** Google Gemini 1.5 Flash
- **Framework:** Google Agent Development Kit (ADK)
- **Language:** Python
- **Tools:** MCP Server, Google Search
- **Platform:** Antigravity IDE
